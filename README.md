# Real Estate App

This is a real estate mobile application I built using React Native to improve my mobile development skills and gain hands-on experience with Firebase, navigation, maps, and responsive UI design.

The app allows users to browse property listings, view property details, search for properties, save their favorite listings, and explore property locations on the map.

## Features

- User authentication with Firebase
- Browse property listings
- Property details screen
- Search functionality
- Save favorite properties
- Google Maps integration
- Responsive user interface

## Tech Stack

- React Native
- JavaScript
- Firebase
- React Navigation
- React Native Maps
- Axios

## Running the Project

Clone the repository:

```bash
git clone https://github.com/mrnoman9n/real-estate-app.git
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
npx expo start
```
